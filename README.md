# Biotime-Attendance-system-using-ExcelVBA
Excel VBA automation system that can process daily ACS attendance report and maintain a historical attendance database, my daily task should only be downloading the ACS reportm selecting the file, and clicking a button to process it.
# Attendance Engine
## Overview
Attendance Engine is an Excel VBA application designed to automate ACS attendance processing.
## Features
- Night Shift Processing
- Morning Shift Processing
- Automatic Punch Out Updates
- Duplicate Prevention
- Historical Database
- Dashboard
- Error Logging
## Performance
Processes 500+ records in under 10 seconds.
