# Biotime Attendance System using Excel VBA

## Project Objective

Build a fully automated Excel VBA attendance system that:

- Imports daily ACS attendance reports
- Processes Morning Shift attendance
- Processes Night Shift attendance
- Prevents duplicate records
- Maintains historical attendance database
- Generates dashboard reporting
